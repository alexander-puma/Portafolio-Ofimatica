# Sección 1: Ofimática en la Nube y Local
Durante el avance de la materia de ofimática, se abordó la edición y optimización de documentos para mejorar su legibilidad y el trabajo colaborativo.  

## Herramientas de Ofimática

Microsoft Word: Empleado para la edición de textos y transformar documentos en textos mejor legibles.  


Google Docs: Utilizado para conocer cómo crear un documento y dividir su elaboración en diferentes partes para crear un mejor resultado.  


Google Drive: Plataforma en la nube utilizada para almacenar y compartir archivos con varias personas.  

# Sección 2: Gestión de Documentos
En esta sección se exploraron métodos eficientes para la manipulación y conversión de archivos digitales.  

## Herramientas de Gestión

ILovePDF: Herramienta considerada de gran utilidad para las siguientes funciones:  

Convertir cualquier tipo de documento, imagen o archivo al formato PDF.  

Unir archivos en un solo documento.  

Gestionar archivos por separado según sea necesario.
# Sección 3: Inteligencia Artificial Generativa
Esta sección describe el uso de modelos de lenguaje avanzados para la optimización de tareas académicas y profesionales.

## Uso de ChatGPT
ChatGPT: Herramienta de IA basada en lenguaje natural que permite potenciar la productividad mediante:

Generación de Contenido: Creación de borradores, resúmenes y explicaciones detalladas sobre temas complejos.

Optimización de Búsquedas: Mejora en la recuperación de información y filtrado de datos relevantes.

Asistencia en Programación y Redacción: Apoyo en la corrección de estilo, traducción de textos y depuración de código.

Interacción Dinámica: Capacidad de mantener un hilo conversacional para refinar respuestas según las necesidades específicas del usuario.
